# Barroco Showdown · 2º ESO

Juego tipo Jeopardy para repasar Barroco europeo, Austrias menores, absolutismo francés, parlamentarismo inglés y Provincias Unidas.

## Archivos

- `index.html`
- `styles.css`
- `script.js`

## Publicación en GitHub Pages

1. Crea un repositorio.
2. Sube estos archivos a la raíz del repositorio.
3. Activa GitHub Pages desde `Settings > Pages`.
4. Elige la rama principal y la carpeta raíz.

## Funcionamiento

- Permite 4 o 5 equipos.
- Guarda la partida en curso en el navegador.
- Guarda historial de últimas partidas.
- Cada casilla usada queda bloqueada.
- La puntuación suma, no resta.
